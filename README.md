### Key Features

| Feature                        | Description                                                                 |
|-------------------------------|-----------------------------------------------------------------------------|
| AI-Powered Assessment       | Predicts mental state using DASS-21 based ML model                         |
| Adaptive Content Delivery    | 3 difficulty levels: Low, Mid, High — served dynamically             |
| Personalized Dashboard       | Progress tracking, SHAP explanations, chapter completion status            |
| Emotion-Aware Chatbot        | Powered by *Google Gemini 1.5 Flash* — responds based on user mood        |
| Full Authentication          | Login, Signup, Forgot Password (with real email reset)                     |
| SHAP Explanations            | Transparent AI — shows which factors influenced the prediction the most    |
| Responsive & Modern UI       | Clean, mobile-friendly design with smooth animations                       |

### Quick Start (Local)

```bash
git clone https://github.com/Asad-Aziz-001/sentellect.git
cd sentellect

python -m venv venv
source venv/bin/activate        # Windows: venv\Scripts\activate

pip install -r requirements.txt

# Download model and place in model/ folder
python app.py
```

Open → http://127.0.0.1:5000

---

### How It Works (Flow)

1. User takes *DASS-21 Survey*  
2. ML Model predicts: Low, Mid, or High stress level  
3. System saves level → serves *customized chapters*  
4. Student completes quizzes → progress updates  
5. AI Chatbot gives *emotional + academic support*
